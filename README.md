# InsightHub – Project Repository & Marketplace

## Project Overview

**InsightHub** is a full-stack web application designed to help students discover, share, and interact with academic projects and ideas.

The platform serves as a **research repository and marketplace**, allowing users to showcase their work, explore other projects, and collaborate within an academic environment.

## ✅ Feature Checklist

### 👨‍🎓 User Features

✔ User registration and login
✔ Secure authentication using JWT
✔ Create and manage project listings
✔ View all available projects
✔ Search and filter projects
✔ Save or express interest in projects
✔ Manage saved projects (bookmark system)
✔ Edit and delete personal submissions
✔ User profile (name, email, skills/interests)

---

### 🔗 Interaction Features

✔ Users can interact with project listings
✔ Bookmark/save system for projects
✔ Status tracking for projects
✔ Feedback alerts for user actions

### 🛠 Admin Features

✔ Role-based access control
✔ Admin dashboard
✔ Ability to delete any project
✔ Monitor platform activity
✔ Manage and moderate listings

### ⚙️ Backend Features

## 💻 Installation Instructions (Run Locally)

### 1. Clone the Frontend Repository

```bash
git clone https://github.com/prebbiebenjamin/insight-hub-frontend.git
cd insight-hub-frontend
```

### 2. Clone the Backend Repository

```bash
git clone https://github.com/prebbiebenjamin/insight-hub-backend.git
cd insight-hub-backend
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Run the Application

```bash
npm start
```
